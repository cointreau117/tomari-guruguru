# トマリ

マウスに追従して25方向に振り向き、自動でまばたきする、トマリ用のブラウザアバターです。
ぐるぐる版とトーク版は1つのアプリに統合済み。さらに「**音楽に反応して踊る**」機能の最小コアを実装済みで、読み込んだ曲のビートに**テンポ同期**してキャラが跳ね、**強い拍では大きく跳ね（跳びポ）**、音符パーティクルを飛ばします。**踊り出すと表情も笑顔に変わります**。

![トマリの動作デモ](guruguru.gif)

## セットアップ

必要環境:

- Node.js 22 LTS 推奨
- Vite 8 の要件: Node.js 20.19+ または 22.12+

```bash
npm install
```

## ローカル起動

Windowsなら `start.bat` をダブルクリックすると、ローカルサーバーを起動してブラウザで開きます。

手動で起動する場合は:

```bash
npm run dev
```

ブラウザが自動で開きます。手動でアクセスする場合は:

```text
http://127.0.0.1:5173/
```

注意:

- マイク入力は `localhost` または HTTPS でのみ利用できます。
- Google Fonts はCDNから読み込むため、初回表示にはネット接続が必要です。

## ビルド

```bash
npm run build
npm run preview   # ビルド結果をローカル確認
```

preview は GitHub Pages と同じ `/tomari-guruguru/` のベースパスで起動します。

```text
http://127.0.0.1:4173/tomari-guruguru/
```

## フレーム画像の仕組み

このアプリは、キャラクターの向きと表情に応じて `public/slices2/` 内の画像を1枚ずつ切り替えています。

### 25方向

5列 × 5行の向き差分です。

- 列: 左向き → 正面 → 右向き
  - `c0`: 左向き / `c1`: 左斜め / `c2`: 正面 / `c3`: 右斜め / `c4`: 右向き
- 行: 上向き → 水平 → 下向き
  - `r0`: 強く上を見る / `r1`: 少し上 / `r2`: 水平 / `r3`: 少し下 / `r4`: 強く下

### 6状態

| フォルダ | 目 | 口 |
|---|---|---|
| `A` | 開け | とじ |
| `B` | 開け | 中間 |
| `C` | 開け | 開け |
| `D` | 閉じ | とじ |
| `E` | 閉じ | 中間 |
| `F` | 閉じ | 開け |

画像パス例: `slices2/A/r2c2.webp`

`src/character-config.js` の `basePath` と `ext` で切り替え可能です。

> 現在のトマリ素材では、踊り中の笑顔表情に **`B`** を流用しています（`B` の中身は「笑顔・目閉じ」に差し替え済みで、上表の「目開け／口中間」とは実態が異なります）。シート構成は将来整理予定です。実際に描画しているのは A（通常）/ D（まばたき）/ B（笑顔）の3枚のみです。

## 使い方

1. アプリ（`index.html` / ローカルなら `http://127.0.0.1:5173/`）を開く
2. マウスを動かすと、キャラクターがマウス方向に合わせて25方向で振り向きます
3. 自動まばたきも入ります（`A`↔`D` の目開け／目閉じを切り替え）
4. 画面下中央の **♪ 音楽ファイルを読み込む** で曲を読み込むと自動再生され、**ビートにテンポ同期して跳ねて音符が飛びます**。曲のテンポにロックするので、低域が薄い区間でもリズムを保って跳ね続けます。**音符の色はテンポに連動**し、速い曲ほど暖色（オレンジ・赤）、遅い曲ほど寒色（青・緑）寄りのパステルになります
5. サビやキメなど**強い拍では、要所で大きく跳ねます（跳びポ）**
6. 踊り出して少し経つと**表情が笑顔に変わります**。ずっと笑顔だと単調なので、適度に通常顔も差し込まれます。静かなブレイクや停止では通常顔に戻ります
7. 別の曲に読み込み直すと、テンポ推定などの内部状態はリセットされ、新しい曲のテンポで仕切り直します
8. 再生音量はプレーヤー（左下、初期50%）で調整。音量を変えても踊りのタイミング・高さは変わりません（音量メーター表示のみ追従）

## Tweaks 調整パネル

画面右下の **Tweaks** ボタンから調整できます。

主な項目:

- ビート感度 / 跳ねる強さ / 強拍アクセントの頻度 / 強拍アクセントの大きさ / 踊り中に笑顔 / 通常顔に戻る頻度 / 音符パーティクル
- 音量感度 / 音量追従の戻り速さ / 自動まばたき
- 追従範囲 / 追従速度 / キャラサイズ / 背景色
- グリッド表示（デバッグ）

OBSなどで使う場合は、背景色をクロマキーしやすい色に調整してください。

パネルは右下の **✕** のほか、**パネルの外側をタップ／クリック**しても閉じられます。

## スマホ表示について

縦長のスマホでも操作できるよう、狭い画面では次のように調整されます。

- 下中央のバーはコンパクトになり、**音量メーターは省略**されます（再生音量はプレーヤー側で調整できます）。
- 曲を読み込むと、ファイル読込ボタンが audio プレーヤーの上に移動して重なりを避けます。
- Tweaks パネルは画面内に収まる高さになり、閉じる **✕** が必ず見える位置に表示されます。

## 自分のキャラで作るには

このアプリで動かすには最終的に **5×5角度シートを6枚** 作る必要があります。

必要な6枚:

```text
A_目開け_口とじ.png
B_目開け_口中間.png
C_目開け_口開け.png
D_目閉じ_口とじ.png
E_目閉じ_口中間.png
F_目閉じ_口開け.png
```

おすすめの流れ:

1. 自分のキャラクター参照画像を用意する
2. `docs/01_画像生成用テンプレ.png` と合わせてChatGPT Images 2.0に添付する
3. `docs/01_画像生成用プロンプト.txt` の指示を使って6枚のシートを作る
4. 6枚のPNGを `新キャラ資料/` フォルダに入れる
5. `tools/slice_character_sheets.py` でスライス画像を生成

詳しい注意点や検証方法は `docs/新キャラ差し替え手順.md` を参照してください。
